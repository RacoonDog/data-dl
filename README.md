# Usage

## Full download
`/data-dl all`

### Advancements
`/data-dl advancement`  
Can only download *known* advancements.

### Recipes
`/data-dl recipe`

### Tags
`/data-dl tag`  
Only the *raw values* are given, references to other tags and optional non-present values are removed.  
Cannot download worldgen-related tags.
